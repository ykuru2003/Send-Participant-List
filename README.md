# 参加者名簿メール送信マクロ

Excelの「参加者名簿」シートから、指定範囲を抜き出して
Outlookで自動的にメール送信するVBAマクロです。

---

## 📋 機能概要
- シート「参加者名簿」から宛先・件名・本文を取得（セル N6～N9）
- セル I3 に書かれた「参加者：○名」から人数を解析
- 該当範囲（A～K列）を一時ファイルに出力
- Outlookを起動し、添付してメールを作成

---

## ⚙️ 使い方
1. Excelファイルを開く  
2. シート「参加者名簿」に以下を入力  
   - N6: 宛先メールアドレス  
   - N7: CC  
   - N8: 件名  
   - N9: 本文  
   - I3: 「参加者：10名」のように人数を含める  
3. `メールで参加者名簿を送信()` を実行

---

## 📁 生成ファイル
- 一時フォルダ（`%TEMP%`）に `部員名簿〇〇.xlsx` を出力します
- メール作成後、自動で削除はしません（必要に応じて手動削除）

---

## 💡 補足
- Outlook がインストールされている必要があります  
- Excelのマクロ実行を許可する設定にしてください
- Windows専用（Mac未確認）

---

## 🧑‍💻 コード構成
- `メールで参加者名簿を送信`: メインの送信処理  
- `GetRowFromJ3`: 「参加者：○名」から人数を取得  
- `CleanFileName`: ファイル名の不正文字を除去
